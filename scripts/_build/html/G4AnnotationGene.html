G4AnnotationGene.getInfo(df)¶

Retrieves informations of a windows and parse it into a dictionary.

As gene windows and junction windows are not formated the same way, this function aims to parse them into the same type of dictionary.

Parameters:df (dataFrame) – contain all overlaping windows.
Returns:dico, contains all infromation for one window.
Return type:dictionary
G4AnnotationGene.mergeG4(df, dicoParam)¶

Browses all junction window to find those that are overlapping.

Here we browse all junctions windows. We will only kept those that overlap the 100 nucleotid. Indeed, if the window over thresholds don’t overlap this position, it only in a gene and not a junction.

Parameters:
  • df (dataFrame) – contains all windows.
  • dicoParam (dictionnary) – contains all parameters that were given to g4rna screener.
Returns:

dfpG4, contain all pG4 for that strand.
Return type:

dataFrame
